QuantStudy Project

Welcome to the QuantStudy project! This project aims to simulate economic indicators over time using advanced computational methods.

Features

  • Economic simulation based on real-world data
  • Visualization of key economic indicators
  • Easy-to-use interface for running simulations

Usage

To run the simulation, follow these steps:

  1. Clone the repository.
  2. Install the dependencies.
  3. Run the main script with the desired parameters.
